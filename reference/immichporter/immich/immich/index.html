    search_assets ¶
    search_assets(
    filename: str | None | Unset = None,
    taken: datetime | str | None | Unset = None,
    taken_before: datetime | str | None | Unset = None,
    taken_after: datetime | str | None | Unset = None,
    **options: any,
) -> list[AlbumResponseDto]

    Search for assets on the Immich server.

    Dates can be formate as follow: Python datetime or string with the format %Y-%m-%d %H:%M:%S or %Y-%m-%d

    Parameters:

    • filename ¶
      (str | None | Unset, default: None ) –

      Filter by filename

    • taken ¶
      (datetime | str | None | Unset, default: None ) –

      Filter by taken date (plus minus 1 day if no time is given, resp. minus 2 hours if no day is given)

    • taken_before ¶
      (datetime | str | None | Unset, default: None ) –

      Filter by taken date before, cannot be used together with taken.

    • taken_after ¶
      (datetime | str | None | Unset, default: None ) –

      Filter by taken date after, cannot be used together with taken.

    • **options ¶
      (any, default: {} ) –

      Additional options, see https://api.immich.app/endpoints/search/searchAssets for more information
